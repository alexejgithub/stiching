# Export/print contents

Type: grilling
Status: resolved

## Question

What should the SVG export/print actually contain — grid only, grid plus numbers, or grid plus numbers plus a color legend?

## Answer

Grid + numbers + legend: the export mirrors the on-screen grid including alternating row/column numbering (see [Numbering convention](04-numbering-convention.md)), plus a legend mapping each palette color+symbol to its label. The whole point of exporting is to take the pattern away from the screen and crochet from a printed sheet, so it must be self-contained — unlabeled colors or missing numbers make it unusable standalone.
