# Grid dimensions

Type: grilling
Status: resolved

## Question

How are pattern dimensions (grid width/height) set — fixed forever at creation, freely expandable, or fixed-with-resize?

## Answer

Fixed at creation via a "New Pattern" dialog (choose rows x columns), with an explicit resize/crop-extend operation available afterward to change dimensions. No infinite/auto-expanding canvas — crochet patterns are planned around a target finished size, but locking dimensions permanently would make a common edit ("3 more rows") impossible without starting over.
