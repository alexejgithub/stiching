# Numbering convention

Type: grilling
Status: resolved

## Question

Are the grid's row/column numbers static (always the same side), or do row numbers alternate sides to reflect crochet's boustrophedon (back-and-forth) reading direction?

## Answer

Alternating by row parity: column numbers run statically along the top, but row numbers flip sides by parity (odd rows numbered on the right, even rows on the left), matching the traditional crochet/knitting chart convention for reading direction. This must be reflected in both the on-screen grid and the SVG export/print (see [Export/print contents](06-export-contents.md)).
